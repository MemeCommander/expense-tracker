# Expense Tracker Web App

A simple, elegant expense tracker built with Python and Flask.

## Features
-   **Dashboard**: View total spent, breakdown by category (Good/Neutral/Bad), and add new expenses.
-   **History**: View full transaction history.
-   **Aliases**: Quick-add categories using persistent aliases.
-   **Themes**: Switch between Dark, Light, Blue, and Red themes (persisted locally).
-   **Undo**: Revert the last action.

## Installation

1.  Clone the repository:
    ```bash
    git clone <your-repo-url>
    cd expense-tracker
    ```

2.  Install dependencies:
    ```bash
    pip install flask
    ```

3.  Run the application:
    ```bash
    python app.py
    ```

4.  Open `http://127.0.0.1:5000` in your browser.
